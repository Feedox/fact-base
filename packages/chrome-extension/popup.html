Savee - Fighting Holocaust Denial on Social Media

Savee is a Chrome extension that allows users to mark text containing false content about the Holocaust on Facebook, Instagram, and Twitter and receive a response with real facts to respond to the post. The extension uses the Feedox API to generate responses based on the selected text.

To use Savee, simply enter the post on Facebook / Instagram / Twitter, highlight the text that contains Holocaust denial, click the right mouse button and select the Savee option. After a few seconds, the comment will appear and you can edit it or immediately publish it.
