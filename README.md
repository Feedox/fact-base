# fact-base
A way to store facts for future fact-checking apps


## Develop
### Setup
1. Clone locally
1. Run `$ yarn install`

### Start:
1. Run `$ yarn start`
2. Open browser on `http://0.0.0.0:3010/`
* Check main package.json for more commands
